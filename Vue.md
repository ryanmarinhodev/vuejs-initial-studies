O vue é divido assim:
<script setup lang="ts">
  // lógica, estado, funções
</script>

<template>
  <!-- HTML com sintaxe Vue -->
</template>

<style scoped>
  /* CSS só desse componente */
</style>


# Vue vindo do React

- `ref` → cria um estado  
  Parecido com `useState`

- `v-for` → renderiza uma lista  
  Parecido com `.map()`

- `:key` → identifica cada item da lista  
  Igual ao `key` no React

- `@click` → reage ao clique  
  Parecido com `onClick`

- `v-if` → mostra ou oculta algo  
  Parecido com renderização condicional no React

- `template` → atualiza a tela quando o estado muda  
  Parecido com o JSX re-renderizando quando o state muda
